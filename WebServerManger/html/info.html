Infomation page

Welcome to the infomation page.
WebServerManger is coded in node.js.
Use the HTTP API this is a basic WebServer.




Configuration

To configure the WebServer goto the "config.json"
This includes the Port, Default Page Name, root directory and the title.
Title: This allows you to change the title on every page.
Root: The folder that contains all the the HTML files
Default Page Name: The file name for /
Port: The port to run the server on

The "files.json" contains all the files.
This currently only has "errorpage".
The error page is the default errorpage for any errors


Directorys

Files (default):
Contains the basic files (Error pages, headers, footers.) (headers and footers can be custom coded if you contact WebServerManger or recode the server.js)

html (default): Contains the websites HTML files
